Вступ до Linux

Linux ([ˈlɪnʊks], укр. Лі́нукс[a]) — загальна назва Unix-подібних операційних систем на основі однойменного ядра. Це один із найвидатніших прикладів розробки вільного (free) та відкритого (з відкритим кодом, open source) програмного забезпечення (software). На відміну від власницьких операційних систем (на кшталт Microsoft Windows та macOS), їхні вихідні коди доступні всім для використання, зміни та поширення абсолютно вільно (в тому числі безкоштовно).

Linux, спершу розроблений для використання окремими ентузіастами на своїх персональних комп'ютерах, пізніше, завдяки підтримці таких компаній, як IBM, Sun Microsystems, HP, Novell та інших, набув неабиякої популярності як серверна операційна система (так, 8 із 10 найбільших компаній, що надають послуги вебхостингу, використовують Linux на своїх вебсерверах).

Лінукс портовано на велику кількість апаратних платформ. Тепер ця ОС досить успішно використовується як на мейнфреймах і суперкомп'ютерах, так і вбудована в багато інших пристроїв (смартфони, планшетні ПК, маршрутизатори комп'ютерних мереж (роутери), пристрої автоматики, системи керування телевізорами та ігровими консолями тощо). Від середини 1990-х Linux все частіше встановлюється і на настільні комп'ютери. Так, станом на березень 2015 його частка складала 1,21 % світового ринку операційних систем на персональних комп'ютерах (без урахування використання на серверах та спеціалізованих пристроях).

Значна кількість спеціалізованих дистрибутивів Linux, які розробляють та підтримують різні спільноти, надає широкі можливості вибору програмного забезпечення.

Популярні дистрибутиви

  • Ubuntu: Найбільш популярний і користувацько-орієнтований дистрибутив. Має великий вибір програмного забезпечення та активну спільноту.
  • Linux Mint: Заснований на Ubuntu, але має більш традиційний інтерфейс, що нагадує Windows. Відмінний вибір для тих, хто переходить з Windows на Linux.
  • Zorin OS: Ще один дистрибутив, який намагається максимально наблизити користувацький інтерфейс до Windows.
  • elementary OS: Має красивий, сучасний інтерфейс, натхненний macOS.
  • Fedora: Орієнтований на нові технології та інновації. Часто використовується для тестування нового програмного забезпечення.
  • Arch Linux: Для досвідчених користувачів, які хочуть мати повний контроль над системою. Вимагає ручної конфігурації.
  • CentOS: Для серверів безкоштовна версія Red Hat Enterprise Linux, популярна для корпоративних серверів.
  • Debian: Стабільний і безпечний дистрибутив для серверів, який часто використовується як основа для інших дистрибутивів.
  • Ubuntu Server: Для серверів стабільна і надійна платформа для серверів.
  • Kali Linux: Спеціалізується на пентестингу та етичному хакінгу.
  • Raspbian: Оптимізований для Raspberry Pi, використовується для створення IoT-пристроїв та інших вбудованих систем.
  • OpenSUSE: Має зручний інструмент для створення інсталяційних дисків з різними конфігураціями.
